几个概念（自己的理解）

元注解：元注解就是jdk中自带的注解，jdk中已经写好了每个注解的处理逻辑(这段代码我在jdk中没找到)，你只要在你写的java代码中使用即可有效。
自定义注解：顾名思义就是你自己去实现一个类似于元注解的注解。
切面：切面就是一个java类
连接点：连接点就是作用在需要作用的方法上的一段逻辑，比如你想要在userInfo()这个方法上加一段日志
